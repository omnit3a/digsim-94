# digsim-94
small mining game using raylib.

# build instructions
## dependencies
- CMake
- Make
- raylib

## building
- `cmake -B .` to generate makefile<br>
- `make` to compile. outputs `game`

# running
## controls
`arrow keys` move the player around<br>
`left-ctrl + arrow keys` rotates the player to face the specified direction<br>
`.` mines blocks
